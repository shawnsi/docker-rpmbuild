From source
-----------

Install the latest development version of docker-rpmbuild on `github
<https://github.com/shawnsi/docker-rpmbuild>`_ by using `pip`: 

.. code-block:: bash

	$ pip install git+https://github.com/shawnsi/docker-rpmbuild

It is recommended to install docker-rpmbuild in a `virtualenv
<http://virtualenv.readthedocs.org/en/latest/>`_ or `pyvenv
<https://docs.python.org/3/library/venv.html>`_  if you are installing from
source.
